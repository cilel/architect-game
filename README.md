![Game Preview](docs/preview.png)

## 🎮 游戏简介

Architect Game 是一款建筑设计类网页游戏，让用户通过**模块化拼装**体验从拿地、规划到内部空间布置的建筑师全流程。

介于**"模拟经营"**（如《模拟城市》）与**"创作工具"**（如《The Sims》）之间，适合建筑爱好者、相关专业学生和休闲模拟游戏玩家。

## ✨ 核心特性

- 🏠 **模块化建造** - 基于网格的体块堆叠系统
- ☀️ **环境模拟** - 实时光照、植被、气候效果
- 📊 **智能评分** - 根据采光、结构、美观度自动评分
- 💰 **预算系统** - 在有限资源下优化设计
- 📸 **拍照分享** - 一键导出精美截图

## 🛠️ 技术栈

- **3D 引擎**: Three.js / Babylon.js
- **前端框架**: React / Vue 3
- **物理引擎**: Cannon.js / Rapier.js
- **后端**: Node.js + Supabase (可选)
- **部署**: Vercel / Netlify

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/cilel/architect-game.git
cd architect-game

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
